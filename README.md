# 🚀 AI Study Copilot

An AI-powered web app that helps students understand their notes using summarization, Q&A, and structured explanations.

---

## 🧠 Features

* ✨ Summarize notes instantly
* 🧒 "Explain like I'm 10" mode
* 💬 Ask questions from your notes
* 🗂️ Generate structured study notes
* 📄 Upload `.txt` files or paste content

---

## ⚡ Tech Used

* HTML + JavaScript (Frontend only)
* Google Gemini API

## 🔑 How to Use

1. Open the website
2. Paste your notes or upload a `.txt` file
3. Enter your Gemini API key
4. Click:

   * Summarize
   * Ask questions
   * Generate notes

---

## 🔐 Get Gemini API Key

1. Go to: https://aistudio.google.com/
2. Click "Get API Key"
3. Copy and paste it into the app

---

## ⚠️ Important

* Do NOT share your API key publicly
* This app runs fully in the browser (no backend)
